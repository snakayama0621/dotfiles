# Aerospace

macOS用のタイルウィンドウマネージャー。i3/sway風のキーバインドで操作。

## 基本設定

| 設定 | 値 |
|------|-----|
| レイアウト | tiles（タイル） |
| ウィンドウ間隔 | 5px |
| 上部マージン | 35px（sketchybar用） |

## キーバインド

### ウィンドウフォーカス

| キー | 動作 |
|------|------|
| `Alt+h` | 左のウィンドウにフォーカス |
| `Alt+j` | 下のウィンドウにフォーカス |
| `Alt+k` | 上のウィンドウにフォーカス |
| `Alt+l` | 右のウィンドウにフォーカス |

### ウィンドウ移動

| キー | 動作 |
|------|------|
| `Alt+Shift+h` | ウィンドウを左に移動 |
| `Alt+Shift+j` | ウィンドウを下に移動 |
| `Alt+Shift+k` | ウィンドウを上に移動 |
| `Alt+Shift+l` | ウィンドウを右に移動 |

### リサイズ

| キー | 動作 |
|------|------|
| `Alt+Shift+-` | ウィンドウを縮小 |
| `Alt+Shift+=` | ウィンドウを拡大 |

### レイアウト

| キー | 動作 |
|------|------|
| `Alt+/` | タイルレイアウト切替（水平/垂直） |
| `Alt+,` | アコーディオンレイアウト切替 |

### ワークスペース

| キー | 動作 |
|------|------|
| `Alt+1-9` | ワークスペース1-9に移動 |
| `Alt+a-g,i,m-z` | ワークスペースA-G,I,M-Zに移動 |
| `Alt+Shift+1-9` | ウィンドウをワークスペース1-9に移動 |
| `Alt+Shift+a-g,i,m-z` | ウィンドウをワークスペースA-G,I,M-Zに移動 |
| `Alt+Tab` | 前のワークスペースに戻る |
| `Alt+Shift+Tab` | ワークスペースを次のモニタへ移動 |

### サービスモード (`Alt+Shift+;`)

| キー | 動作 |
|------|------|
| `Esc` | 設定リロードしてメインモードへ戻る |
| `r` | レイアウトリセット |
| `f` | フローティング/タイリング切替 |
| `Alt+Shift+h/j/k/l` | ウィンドウ結合（左/下/上/右） |

## 連携

- **sketchybar**: ワークスペース変更時に通知（`exec-on-workspace-change`）
